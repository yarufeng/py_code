TWiki Registration

Account creation is automated on Twiki, if you have group access and you are not able to access the site, please create a ticket and send an email to webapps@amd.com.

I have an account, but cannot access a page or web

-If you have reached this page due to an access issue and need to request access, please follow these steps:

Copy paste the twiki url here: http://twiki.amd.com/misc/check-permissions

This should display the group that you need to be part to get access.

1.Design Group: Servicenow link

2. Unix group/other groups: https://techprotect.amd.com/access/user_control/

Please note: On techprotect site the group name will be different for eg. CORESDOC should be searched with key word cores, you can select the group matching your twiki group eg CORESDOC will displayed as cores_doc. In case NDA required please complete or else access won't be permitted.

3. Legacy groups are not present on servicenow or techprotect use this link to request legacy groups : http://mpdwww.amd.com/cgi-bin/request/

Check your Twiki/webauth group access here: http://mpdwww.amd.com/ec/ec_audit.php

Please note that it can take up to half an hour from notification for your account to be fully configured in the TWiki.
